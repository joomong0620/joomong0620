# 💿 Intro


> ## 더 큰 세상을 향해 나아가는 개발자, 김주연입니다. 
> 저는 현재에 머물러 있기보다, 한 단계 더 넓은 시야와 더 깊은 기술력을 갖춘 개발자로 성장하고 싶습니다.
빠르게 변화하는 기술 환경 속에서 새로운 개념을 배우고, 이를 직접 구현해보는 과정을 즐깁니다. <br />

> **Python**을 기반으로 **FastAPI·Django** 백엔드 개발과 AI 프로젝트를 경험하며,
단순히 코드를 작성하는 것이 아니라 **문제를 구조적으로 해결하는 방법**을 고민해왔습니다.
특히 AI 모델과 서버를 연동하고, 실제 사용 환경에서 피드백을 반영해 개선하는 과정을 통해
기술은 결국 사람을 위해 존재한다는 점을 배우게 되었습니다.  <br />


> 저는 ‘잘 만드는 개발자’를 넘어,
**더 넓은 세상에서 통하는 시스템** 을 설계할 수 있는 개발자로 성장하고 싶습니다. <br />
>또한 긍정적인 마인드로 팀원들에게 힘이 되는 사람으로 남고 싶습니다.
함께 고민하고, 함께 배우고, 함께 완성해가는 과정 속에서 가장 큰 보람을 느낍니다.
